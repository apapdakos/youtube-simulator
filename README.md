# YouTube Simulator

## 🚀 Overview
A desktop application that simulates the core functionality of a video platform. The project focuses on managing video content, user interaction models, and data persistence using Java.

## 🛠️ Tech Stack
* **Language:** Java
* **Concepts:** Object-Oriented Programming, Data Structures, File I/O.

## 🔑 Key Features
* **Video Management:** Create, delete, and organize video entries.
* **Interaction Simulation:** Track likes, views, and simulated user activity.
* **Data Persistence:** Save and load platform state using local file storage.

## 🎓 Academic Context
This project was developed for the OOP Java Programming Class module at the Technical University of Crete (TUC), achieving a grade of 7/10.
